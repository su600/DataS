* Python-snap7
* Pylogix
* Pyads
* PyModbus
* OPC UA  62541
* UR

* [ ] BECKOFF PLC
* [ ] OPC UA devices.
* [ ] Modbus
* [ ] UR Robot
* [ ] Kuka Robot
