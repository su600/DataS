> **This is beta version.**

Flask based industrial SCADA application.

Base on great works of

* Python-snap7
* Pylogix
* Pyads
* PyModbus
* OPC UA  62541
* UR

For

* [x] SIEMENS PLC
* [x] ROCKWELL  AB PLC
* [x] BECKOFF PLC
* [x] OPC UA devices.
* [x] Modbus
* [ ] UR Robot
* [ ] Kuka Robot
